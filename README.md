Projektbeskrivning:
SupportMe är en desktopapplikation som går ut på att hjälpa användaren med diverse tekniska vardagsproblem.
 
1. För att starta programmet kör metoden "Main" i mappen "controller"
2. De initiala användarnamnet för administratörsrättigheter är admin. Lösenorder är identiskt med användarnamnet.
3. För att kunna logga in som användare krävs det att du anger ett användarnamn, gilltig epost samt ett lösenord.


Länk till källkoden finns på [GitHub](https://github.com/PhPersson/DA336A-Projekt/tree/RS2/src)
